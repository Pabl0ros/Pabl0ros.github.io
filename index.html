NOSOTROS

En las clases de Laboratio de diseño web utilizamos los siguientes lenguajes.

HTML 5
1° Lenguaje

HTML5 es un estándar que define una estructura básica y un código para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros.

CSS 3
2° Lenguaje

El CSS es un lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML.

Java Script
3° Lenguaje

JavaScript es un lenguaje que se utiliza para hacer páginas web interactivas. Las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.
